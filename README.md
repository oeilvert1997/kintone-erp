# kintone-erp

An ERP system built with Bun, Vite, and SolidJS for kintone customization.

## Prerequisites

This project uses Infisical for secret management.
Before running any scripts, ensure you are authenticated:

```bash
infisical login
```

## Getting Started

```bash
bun install
```

## Sync kintone app schemas

```bash
bun gen:types
```

## Local Development

```bash
bun run dev
```
